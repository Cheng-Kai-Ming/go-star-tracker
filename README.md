# Go Star Tracker

A simple command-line application built with Go.

## Usage
```bash
go run app.go
```

Input the GitHub username, and the app will output the number of stars in markdown format.

## Example
```bash
> Cheng-Kai-Ming
# GitHub Stars429006
**********************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************************....
```
